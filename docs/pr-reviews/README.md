# PR Review Artifacts

This folder stores curated markdown copies of AI PR review outputs for traceability.

## Naming Convention

- `pr-<number>-<slug>-review.md`

Examples:
- `pr-23-smoke-run-review.md`
- `pr-42-data-pipeline-review.md`

## Notes

- Workflow runtime artifacts are always available in GitHub Actions artifact downloads.
- Only meaningful/curated review markdown files should be committed here.
- Temporary trigger notes or duplicate raw outputs should be removed after validation.
