Official OpenCode CLI Documentation Reference:
===============================================

Command: opencode run

Usage: opencode run [options] "<prompt>"

Options:
  --agent <name>           Specify which agent to use
  --model <provider/model> Override default model
  --format <default|json>  Output format
  --continue                Continue existing session
  --session <id>           Resume specific session ID
  --file <path>            Attach file to context

Invalid Flags (NOT supported):
  ❌ --input (does not exist)
  ❌ --output (does not exist)

Output Capture Methods:
1. Shell redirection: opencode run --agent foo "prompt" > output.md
2. JSON format: opencode run --agent foo "prompt" --format json > output.json
